## ObjectScale

{{<table "table table-striped table-bordered table-sm">}}
| Area              | Core Features          |  Details |
|:------------------|:-----------------------|-----------------------------------------------------------------------------------------------------------|
| Provisioning      | _Create Bucket_        | <div style="text-align: left"> Bucket is created using default settings.                                  |
|                   | _Use Existing Bucket_  | <div style="text-align: left"> Bucket is created based on existing bucket in ObjectScale.                 |
|                   | _Delete Bucket_        | <div style="text-align: left"> Bucket is deleted.                                                         |
| Access Management | _Grant Bucket Access_  | <div style="text-align: left"> Full access is granted for given bucket.                                   |
|                   | _Revoke Bucket Access_ | <div style="text-align: left"> Access is revoked.                                                         |
